# Dijkstra
This repository is for studing Dijkstra algorithm.

### How to use
1. Download this repository to your folder
2. open ".sln" file on visual studio

or
```bash
$ git clone https://github.com/Holmes1st/Dijkstra.git
$ gcc -o OUTPUT_NAME example.c djikstra.c graphFunc.c priorityQueue.c
```
or just
```bash
$ make
```
